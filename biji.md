# 搭建个人博客
+ 在本地写Markdown
+ 写完push到仓库
+ 个人博客自动得到工薪
+ 也可以配置自己的专属域名
+ 打开个人GitHub
  * 创建一个仓库
  * 把本地页面推到GitHub上
  * 支持托管静态页
  * setting
  * GitHubpages
  * master branch  save
  * 这种方式必须作为子级目录才能访问到野蛮
  * 如何通过https://你的GitHub来访问个人博客的域名
  * 仓库名叫  github用户名.github.io
  * 对于github用户名.io仓库,github会默认开启github pages功能.只要里面有一个index.html的文件,
  * 那我们就可以通过域名来访问我们的个人博客
+ hexo
  * 让Markdown转换为静态的网页
